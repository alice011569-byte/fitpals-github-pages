# FitPals

FitPals is a prototype fitness reward app. Users can collect cute characters, claim rewards from workouts, grow characters with items, decorate a character room, and view monthly reward history.

## GitHub Pages setup

1. Create a new GitHub repository, for example `fitpals`.
2. Upload every file in this folder to the repository root.
3. Go to `Settings > Pages`.
4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save.
6. Wait until GitHub shows the published URL.

The final URL will look like:

```text
https://YOUR_GITHUB_USERNAME.github.io/fitpals/
```

Open that URL on iPhone Safari and use `Share > Add to Home Screen`.
